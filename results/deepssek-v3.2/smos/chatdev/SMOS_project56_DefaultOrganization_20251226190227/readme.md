# Teacher Class List Viewer
## Description
This application implements the ViewClassListTeacher use case, allowing teachers to view the list of classes they teach for selected academic years.
## Features
- Displays academic years in which the teacher has classes
- Allows selection of academic year
- Shows classes for the selected year
- Simulates connection to SMOS server with random interruptions
## Prerequisites
- Java JDK 8 or higher
- Command line interface
