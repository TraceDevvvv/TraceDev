MODIFY BANNER JAVA PROGRAM
==========================

This Java program implements the "ModifyBanner" use case for changing the image of a banner ad.
The program demonstrates the complete flow described in the use case specification.

PROGRAM STRUCTURE
-----------------
The program consists of 5 Java classes:

1. Banner.java
   - Represents a banner with an image and associated metadata
   - Handles banner data including image file validation
   - Methods: validateImage(), updateImage(), backupImage(), etc.

2. RestPoint.java
   - Represents a refreshment point (rest point) with associated banners
   - Manages collection of banners at a specific location
   - Methods: addBanner(), removeBanner(), updateBannerImage(), validateAllBanners(), etc.

3. AgencyOperator.java
   - Represents an agency operator who can modify banner images
   - Handles the complete banner modification flow as per the use case
   - Methods: login(), executeModifyBannerFlow(), validateImageForBanner(), etc.

4. ImageValidator.java
   - Comprehensive image validator for banner advertisement images
   - Performs checks for format, size, file signature, and basic validation
   - Includes ValidationResult inner class for detailed validation reports

5. Main.java
   - Main class demonstrating the complete ModifyBanner use case
   - Creates sample data, initializes AgencyOperator, runs banner modification flow
   - Demonstrates error cases and server interruption handling

USE CASE IMPLEMENTATION
-----------------------
The program implements all steps of the ModifyBanner use case:

1. Receives list of rest points (from SearchRefreshmentPoint use case) and selects one
2. Views list of banners associated with the selected rest point
3. Selects a banner from the list and enters editing functionality
4. Displays form for selection of an image
5. Selects picture and sends request to change the system
6. Checks characteristics of inserted image; if invalid, enables Errored use case
7. Confirmation of transaction change
8. Bookmarks new image for the selected banner

Exit condition: System notifies successful modification of the banner
Handles: Interruption of connection to server ETOUR

COMPILATION AND EXECUTION
-------------------------

EDGE CASES HANDLED
------------------
1. Invalid image files (non-existent, wrong format, too large)
2. Server interruption (ETOUR connection loss)
3. Invalid user input (non-numeric selections, out of range)
4. File permission issues
5. Empty or null image paths
