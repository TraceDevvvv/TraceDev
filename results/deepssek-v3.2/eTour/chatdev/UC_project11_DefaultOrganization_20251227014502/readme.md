# Convention History Viewer Application
## Overview
A Java Swing application that displays convention history for selected restaurants (points of rest). Uses MVC architecture with H2 in-memory database for immediate runnability.
## Features
- View convention history for specific restaurants
- Automatic database setup with sample data
- Connection status monitoring
- Fallback to in-memory database if MySQL unavailable
- User-friendly GUI with error handling
## Requirements
- Java 8 or higher
- No external database setup required
